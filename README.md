# CondaPackaging

A repo for testing conda packaging using various compilers and methods.

## Goal
Improve understanding of the c++ compilation process and linking with python code.

## Packages subject for testing

* gmsh
* pythonocc-core
* occt
* fltk
* freetype
* ifcopenshell
